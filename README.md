LaTeXBuilder for Codiad
========================
This plugin adds a LaTeX builder to [Codiad](https://github.com/Codiad/Codiad) 
which allow to build PDF-files.
The PDF viewer has `SyncTeX` included to analyse the output PDF quickly.
